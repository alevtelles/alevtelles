# Alexsander Telles

Sou Arquiteto de Soluções em IA e Engenheiro de Software, com mais de 15 anos de experiência projetando plataformas escaláveis, arquiteturas corporativas e produtos baseados em Inteligência Artificial.

Meu trabalho é transformar Inteligência Artificial Generativa em soluções prontas para produção, combinando Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), Sistemas Multiagentes, arquiteturas modernas de backend e aplicações cloud-native.

Tenho experiência na construção de plataformas corporativas de IA que equilibram arquitetura de software, escalabilidade, governança, observabilidade e geração de valor para o negócio. Atuo desde a definição da arquitetura até a implantação em produção, liderando decisões técnicas, desenvolvendo soluções complexas e apoiando equipes de engenharia.

Participei do desenho e implementação de uma plataforma corporativa de Inteligência Artificial voltada à construção de assistentes inteligentes para ambientes empresariais.

A solução combinava Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), agentes inteligentes e práticas modernas de engenharia de software para entregar aplicações seguras, escaláveis e preparadas para produção.

### Principais capacidades

* Arquitetura Multiagente
* Retrieval-Augmented Generation (RAG)
* Busca Semântica
* Gestão de Conhecimento Corporativo
* Orquestração de IA com LangGraph
* Backend desenvolvido em FastAPI
* Camada de abstração para múltiplos provedores de LLM
* Versionamento de Prompts
* Avaliação de modelos
* Observabilidade de IA
* Guardrails e Governança
* Arquitetura Multi-Tenant
* APIs REST
* Integração com sistemas corporativos

Esse projeto representa minha experiência no desenvolvimento de plataformas corporativas de IA, onde arquitetura, governança, escalabilidade e manutenção são tão importantes quanto o próprio modelo de linguagem.
